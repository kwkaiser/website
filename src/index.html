About

Who am I?

Hey, I'm Karl. Enthusiast for data science, software, bioinformatics, linux, and whatever else is drawing my attention this week. I made this site to act as a poster for some of my work and to help me keep track of some of my projects.


Quick background

I'm a graduate of the University of Vermont, with a M.S. in computer science and a B.S. in neuroscience. Some of my previous work includes:

  • Wildfire border prediction via deep learning models & cellular automata
  • Evolution of modular networks via genetic algorithms
  • Natural language processing to generate metrics of aggression in twitter data
  • Database administration / schema design
  • API construction
  • Deploying microservice systems via Kubernetes
  • Building automated data cleaning / processing pipelines
  • Much more
